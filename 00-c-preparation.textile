** Test at "http://localhost:8301/cfide/administrator/":http://localhost:8301/cfide/administrator/
** Path should be C:\JRun4\servers\cfwheels101\

h3. Install Apache ( optional )

* Apache
** Download and install the "Win32 Binary without crypto (no mod_ssl) (MSI Installer)":http://apache.tradebit.com/pub//httpd/binaries/win32/httpd-2.2.17-win32-x86-no_ssl.msi
** We will modify three files: httpd.conf, httpd-vhosts.conf, and hosts

First uncomment these lines by deleting the hash @#@ in front of the line and save the file.

* C:\Program Files (x86)\Apache Software Foundation\Apache2.2\conf\httpd.conf

<pre lang="cfm">
<code>
LoadModule proxy_module modules/mod_proxy.so
LoadModule proxy_http_module modules/mod_proxy_http.so
Include conf/extra/httpd-vhosts.conf
</code>
</pre>

Next add this code to the bottom the the next file, @httpd-vhosts.conf@.

* C:\Program Files (x86)\Apache Software Foundation\Apache2.2\conf\extra\httpd-vhosts.conf

<pre lang="cfm">
<code>
<VirtualHost *:80>
 ServerName wheels.local

 ProxyRequests Off
  <Proxy *>
  Order deny,allow
  Allow from all
  </Proxy>

 ProxyPass / http://wheels.local:8301/
 ProxyPassReverse / http://wheels.local:8301/
</VirtualHost>
</code>
</pre>

Finally, open the last file, @hosts@, and add this line at the top.

* C:\Windows\System32\drivers\etc\hosts add this:

<pre lang="cfm">
<code>
127.0.0.1 	wheels.local
</code>
</pre>

Now instead of going to "http://localhost:8301/":http://localhost:8301/, you can get to our ColdFusion server instance via "http://wheels.local/":http://wheels.local/.
