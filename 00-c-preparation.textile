h1. Preparation for JSBloggers Project

There are a few extra steps of preparation when working with CFWheels projects:

h3. Installing Servers

h4. Install Adobe ColdFusion 9

* Adobe ColdFusion 9
** Download and install "http://www.adobe.com/products/coldfusion/":http://www.adobe.com/products/coldfusion/ as multiple instances
** Name the instance **cfwheels101**
** Test at "http://localhost:8301/":http://localhost:8301/

h3. Install Apache ( optional for http://wheels.local/ )

* Apache
** Download and install the "Win32 Binary without crypto (no mod_ssl) (MSI Installer)":http://apache.tradebit.com/pub//httpd/binaries/win32/httpd-2.2.17-win32-x86-no_ssl.msi
** We will modify three files ( httpd.conf, httpd-vhosts.conf, and hosts ).

First uncomment these lines by deleting the hash in front of the line and save the file.

* C:\Program Files (x86)\Apache Software Foundation\Apache2.2\conf\httpd.conf

<pre>
<code>
LoadModule proxy_module modules/mod_proxy.so
LoadModule proxy_http_module modules/mod_proxy_http.so
Include conf/extra/httpd-vhosts.conf
</code>
</pre>

Next add this code to the bottom the the next file.

* C:\Program Files (x86)\Apache Software Foundation\Apache2.2\conf\extra\httpd-vhosts.conf

<pre>
<code>
<VirtualHost *:80>
 ServerName wheels.local

 ProxyRequests Off
  <Proxy *>
  Order deny,allow
  Allow from all
  </Proxy>

 ProxyPass / http://wheels.local:8301/
 ProxyPassReverse / http://wheels.local:8301/
</VirtualHost>
</code>
</pre>

Finally, open this file and add this line.

* C:\Windows\System32\drivers\etc\hosts add this:

<pre>
<code>
127.0.0.1 	wheels.local
</code>
</pre>

Now instead of you should get to our ColdFusion server via "http://wheels.local/":http://wheels.local/ .

h4. Install ColdFusion on Wheels

* ColdFusion on Wheels
** Download "http://cfwheels.org/download/latest-version":http://cfwheels.org/download/latest-version
** Extract and copy the file to "C:\JRun4\servers\cfwheels101"
** Test at "http://wheels.local/":http://wheels.local/

h4. Install DBMigrate Plugin

* DBMigrate
** Download "http://code.google.com/p/cfwheels-dbmigrate/":http://code.google.com/p/cfwheels-dbmigrate/
** Move the downloaded zip into "C:\JRun4\servers\cfwheels101\plugins"
** Reload CFWheels "http://wheels.local/":http://wheels.local/
** You should see DBMigrate under Plugins

h4. Install Scaffold Plugin

* Scaffold
** Download "http://cfwheels.org/plugins/listing/9":http://cfwheels.org/plugins/listing/9
** Move the downloaded zip into "C:\JRun4\servers\cfwheels101\plugins"
** Reload CFWheels "http://wheels.local/":http://wheels.local/
** You should see Scaffold under Plugins
