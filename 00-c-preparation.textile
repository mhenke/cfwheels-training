h1. Preparation for JSBloggers Project

There are a few extra steps of preparation when working with CFWheels projects:

h3. Installing Servers

h4. Install Tomcat

* Tomcat
** Download and install "32-bit/64-bit Windows Service Installer":http://www.trieuvan.com/apache/tomcat/tomcat-7/v7.0.8/bin/apache-tomcat-7.0.8.exe
** Test at "http://localhost:8080/":http://localhost:8080/

h4. Install Adobe ColdFusion 9

* Adobe ColdFusion 9
** Download and install "http://www.adobe.com/products/coldfusion/":http://www.adobe.com/products/coldfusion/ as a war
** Rename cfusion.war to cf9.war
** Copy cf9.war to "C:\Program Files\Apache Software Foundation\Tomcat 6.0\webapps\"
** Test at "http://localhost:8080/cf9/":http://localhost:8080/cf9/

h4. Install ColdFusion on Wheels

* ColdFusion on Wheels
** Download "http://cfwheels.org/download/latest-version":http://cfwheels.org/download/latest-version
** Extract and copy the file to "C:\Program Files\Apache Software Foundation\Tomcat 6.0\webapps\cf9"
** Test at "http://localhost:8080/cf9/":http://localhost:8080/cf9/

h4. Install DBMigrate Plugin

* DBMigrate
** Download "http://code.google.com/p/cfwheels-dbmigrate/":http://code.google.com/p/cfwheels-dbmigrate/
** Move the downloaded zip into "C:\Program Files\Apache Software Foundation\Tomcat 6.0\webapps\cf9\plugins"
** Reload CFWheels "http://localhost:8080/cf9/":http://localhost:8080/cf9/
** You should see DBMigrate under Plugins

h4. Install Scaffold Plugin

* Scaffold
** Download "http://cfwheels.org/plugins/listing/9":http://cfwheels.org/plugins/listing/9
** Move the downloaded zip into "C:\Program Files\Apache Software Foundation\Tomcat 6.0\webapps\cf9\plugins"
** Reload CFWheels "http://localhost:8080/cf9/":http://localhost:8080/cf9/
** You should see Scaffold under Plugins

h4. Install MySQL Server

* Windows
** Download and install "http://dev.mysql.com/downloads/mirror.php?id=401202":http://dev.mysql.com/downloads/mirror.php?id=401202
