JSBlogger

In this project you'll create a simple blog system and learn the basics of ColdFusion on Wheels including:

  • Models, Views, and Controllers ( MVC )
  • Data Structures & Relationships
  • Routing
  • Migrations
  • Views with forms, partials, and helpers
  • RESTful design
  • Using Wheels plugins

The project will be developed in five iterations below. The completed iterations are zip files located at https://github.com/mhenke/cfwheels-training/ :

I0: Up and Running

I1: Form-Based Workflow

I2: Adding Comments

I3: Tagging

I4: Installing Plugins

I5: Authentication

I6: Extras
